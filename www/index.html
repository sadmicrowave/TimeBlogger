Creating Projects

To create a project, click on the '+' button in the project window, enter the project name, and click 'save'.

Editing Projects

To edit a project name, click on the edit button in the project window, select a project to edit, then change the project name and click 'save'.

Deleting Projects

To delete a project, click on the edit button in the project window, click the delete icon for the project you wish to delete and press the resulting delete button. Alternatively, you can swipe your finger across the project name and tap the resulting delete button. Note, deleting a project automatically deletes all associated tasks.

Creating Tasks

To create a task within a project, click on the project you want to enter, then click the '+' button in the task window. Enter a task name and some details, and start the timer. When your ready, click 'save'.

Updating a Task

To edit/update a task click on a task in the task window, change/update any information you wish and tap 'save'.

Task Status

Task Status is a way to keep track of tasks that have not been started, are currently in process, or have been completed. To change the task status click on a task in the task window and tap any of the three task status options in the segmented control bar at the top of the task details window.

Deleting a Task

To delete a task swipe your finger across the task name and click the resulting delete button. Alternatively, you can click a task in the task window, scroll to the bottom of the task details window and press the 'Delete This Task' button.

Sorting Lists

You can sort project and task lists by utilizing the sorting segmented control bar at the top of each list window. Tap each section multiple times to switch between ascending and descending sort orders.
